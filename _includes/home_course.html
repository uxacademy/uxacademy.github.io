UX Workshop

เน้นความเข้าใจในขั้นตอนการออกแบบประสบการณ์ของผู้ใช้ ผู้เรียนจะได้เรียนรู้เทคนิคการทำ Customer Development การเขียน User Journey และการทำ Usability Test เพื่อที่จะออกแบบโปรแกรมได้ตรงกับใจของผู้ใช้มากที่สุด

UI Workshop

งานออกแบบ UI ไม่ใช่แค่เรื่องสวยงาม ต้องคำนึงถึงความง่ายในการทำความเข้าใจ ถ้าใช้เวลาในการเข้าใจ UI น้อยก็จะมีเวลาในการเข้าใจเนื้อหาของโปรแกรมได้มากขึ้น เราจะมาเรียนทั้งเรื่อง Design Principle, Wireframe, Information Architecture และนำ UI ต่าง ๆ มาลองปรับแก้กันดู
