บริษัท TAT (The Astonishing Tribe) ได้ทดลองออกแบบ Mobile UI โดยเลียนแบบลักษณะของงานสิ่งพิมพ์ ได้ผลมาตาม video ครับ

ผมว่าแนวทางนี้น่าสนใจมากทีเดียวครับ ก่อนหน้านี้ตอนที่เราเริ่มทำเว็บกันใหม่ๆ เราก็พยายามเอาหน้าสิ่งพิมพ์มาไว้บนเว็บเช่นเดียวกัน จนสุดท้ายเว็บก็มีแนวทางของมันเอง

แต่สำหรับโลก Mobile จะไม่เหมือนกัน ด้วยลักษณะการถือการอ่าน จะคล้ายกับสิ่งพิมพ์พวกหนังสือ หรือนิตยสารมากกว่า ดังนั้นมันอาจจะประสบความสำเร็จก็ได้

ผมเข้าใจว่าโลกของสิ่งพิมพ์มันวิวัฒนาการมานานมาก มี best practist อยู่มากมายที่เอามาใช้ได้ทันที ถ้าเราเปิดใจเราจะสามารถรับเอาสิ่งดีๆ เหล่านี้มาไว้ใน Application ของเราได้ครับ
