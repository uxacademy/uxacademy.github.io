จากครั้งที่แล้วได้รีวิว tools สำหรับใช้ทำ usability testing ไป งวดนี้ขอมาเสนอ Tools สำหรับใช้ทำ Interactive Prototype บนมือถือเพิ่มอีก 3 ตัวนะคะ

1. Prototyping On Paper (POP) - http://popapp.in/

ราคา: ฟรี

POP เป็นแอพเล็กๆแต่ไอเดียโดนใจ เพราะวิธีใช้ง่ายมากแค่เสก็ตช์รูปหน้าจอมือถือแต่ละหน้า ใช้มือถือถ่ายรูปเสก็ตช์เอาไว้ แล้วเพิ่มลิงค์ไปยังหน้าต่างๆเพื่อสร้าง flow

ความเจ๋งของแอพนี้อยู่ที่เราสามารถเซฟมันไว้ในเครื่องแล้วพกเอาไปเทสที่ไหนก็ได้ ให้ฟีลเหมือนใช้งานจริงๆ

ข้อดี: ใช้ง่าย, ได้ prototype เร็ว, เทสบนมือถือได้

ข้อเสีย: บันทึก interaction ของผู้ใช้ไม่ได้, ใช้ gesture หรือเอฟเฟกต์ต่างๆไม่ได้, ตอนนี้ใช้ได้แค่บน iPhone

 

2. Proto.io - http://proto.io

ราคา: ฟรี ถึง $49 ต่อเดือน

Proto.io ก็เป็นเหมือนรุ่นพี่ของPOP ถ้าPOPเอาไว้ทำ low-fidelity prototype (prototype ความละเอียดต่ำ) Proto.io ก็เอาไว้ทำ prototype ความละเอียดสูง โดยวิธีใช้ก็ง่ายๆเหมือนกันค่ะ เราสามารถสร้างหน้าจอมือถือได้จากการลากและวาง component ที่เค้าเตรียมไว้ให้ เช่น tab รูปภาพ ปุ่ม และสามารถกำหนด transition ต่างๆได้ด้วย เช่น เอฟเฟกต์ตอนสลับหน้าจอ เอฟเฟกต์ตอนหมุนหน้าจอเป็นแนวนอน เมื่อสร้างเสร็จเราสามารถเก็บ prototype ไว้ในมือถือเพื่อใช้ในการเทสได้เช่นกันค่ะ

รูปด้านซ้ายคือตัวอย่าง prototype ที่ทำเสร็จแล้ว

ข้อดี: ใช้ได้ทั้ง Android และ iPhone และ tablet, สร้าง prototype ได้สมจริงมาก, เทสบนมือถือได้

ข้อเสีย: ราคาสูงพอสมควร, บันทึก interaction ของผู้ใช้ไม่ได้, ใช้ gesture ไม่ได้

 

 

 

 

 

 

 

3. SolidifyApp - http://www.solidifyapp.com

ราคา: $19 ถึง $49 ต่อเดือน

SolidifyApp เป็นแอพที่รวมทั้ง prototyping และ usability testing เข้าด้วยกัน แอพจะให้เราอัพโหลดหน้าเพจต่างๆ และลิงค์เข้าด้วยกัน หลังจากนั้นเราสามารถดาวน์โหลดลงมือถือ tablet เพื่อนำไปเทสได้ โดยจะเป็นเหมือน one stop service มีบริการหา tester มาเทสให้เราด้วย หลังจากเทสเสร็จโปรแกรมก็จะบันทึก interaction ของผู้ใช้ และสรุปผลให้ ทำให้สามารถวิเคราะห์ข้อมูลได้ง่ายขึ้น

ข้อดี: รวมความสามารถทุกอย่างไว้ด้วยกัน เทสได้ทุก platform บันทึกข้อมูลได้ สรุปข้อมูลได้

ข้อเสีย: ราคาสูงพอสมควร และใช้ gesture ไม่ได้

 

สรุปง่ายๆว่าถ้าอยากสร้างprototypeเบืื้องต้นให้ลองใช้POP ถ้าอยากทำแบบละเอียดนิดนึงก็ Proto.io แต่ถ้าอยากทำทุกอย่างที่ว่ามาก็ใช้ Solidify เลยค่ะ

หวังว่าจะเจอ tools ที่ถูกใจบ้างนะคะ ถ้ามี tools ตัวอื่นที่เด็ดๆก็เอามาแชร์กันได้ค่ะ
