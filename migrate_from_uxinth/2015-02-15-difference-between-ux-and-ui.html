กลับจากเดินทางไปเป็นที่ปรึกษางานพัฒนาเว็บไซต์และโปรแกรมบนมือถือของศูนย์คอมพิวเตอร์ ม.ขอนแก่น มาครับ ได้เรียนรู้อะไรมากมาย ทั้งการออกแบบเพื่อกลุ่มผู้ใช้ที่เป็นนักศึกษาและการออกแบบเพื่อกลุ่มผู้ใช้ที่เป็นเจ้าหน้าที่ ได้ร่วมมือกับนักพัฒนาของศูนย์คอมฯ ทดสอบโปรแกรมผ่านกระดาษ เพื่อลดเวลาในการลองผิดลองถูก ซึ่งทำให้เราพบอะไรใหม่ๆ ทุกครั้งที่ได้ทดลองครับ

แต่สิ่งที่อยากมาเล่าวันนี้เป็นภาพที่น้องที่เมื่อก่อนเคยทำงานด้วยกันเอามาให้ดูแล้วบอกว่าพี่นี่ละเป็นคนส่งให้ ผมดูแล้วก็คิดว่าน่าจะเอามาแลกเปลี่ยนกันครับ

มันเป็นภาพที่อยู่ใน blog design.org ซึ่งพูดถึงความแตกต่างระหว่าง UX และ UI โดยเอาไปเปรียบเทียบกับอาหารเช้าซีเรียล เริ่มต้นจากคำว่า Product

คำว่า Product หมายถึงวัตถุดิบทุกอย่างที่จะเอามาประกอบกันเป็นอาหารเช้าให้เรากิน ไม่เฉพาะตัวซีเรียล แต่รวมถึงจานช้อน นม ถ้วยใส่นม ด้วย 

ส่วน UX ก็หมายถึงตอนที่ทุกอย่างประกอบเข้าด้วยกันแล้ว ผู้ใช้ได้เห็นว่ามีนมราดอยู่บนซีเรียล ไม่มากจนล้น ไม่น้อยจนรู้สึกว่าจะไม่อิ่ม มีช้อนเสียบลงไปในถ้วย ผู้ใช้เห็นแล้วก็รู้ว่าจะต้องจับช้อนแล้วลองตักเข้าปาก เคี้ยวกร้วมๆ รู้สึกสัมผัสถึงความกรอบของซีเรียล และความนุ่มข้นของนม ความสนุกที่ได้ตักอีกช้อน และตอนที่กวาดนมหยดสุดท้ายเข้าปาก แล้วรู็สึกอิ่มอกอิ่มใจกับอาหารเช้าจานนี้ นี่ละ Experience

<

p style="text-align: left;">UI คือส่วนประกอบที่ผู้ใช้ต้องใช้เพื่อทำงานเสร็จ เมื่อผู้ใช้เห็นช้อนก็รู้ว่าจะต้องใช้งานมันอย่างไร ผู้ใช้สามารถสร้างงานได้จาก User Interface ของเราซึ่งในที่นี้ก็คือการใช้มันตักซีเรียลเข้าปาก เทียบได้กับการใช้โปรแกรม preview เปิดรูปภาพครับ จะเห็นว่าถ้าไม่มี Interface ที่ดีการที่จะได้ Experience ที่ดีก็เป็นไปได้ยากและอาจไปถึงขั้นทำให้งานกร่อย ซีเรียลไม่อร่อย  รูปภาพไม่สวยไปเลย
Server เทียบได้กับคลังข้อมูลหรือส่วนที่เป็นคุณค่าของโปรแกรม ถ้าให้ถูกต้องก็ต้องมีกล่องนมวางไว้ด้วย เมื่อผู้ใช้ทานซีเรียลจานแรกหมดแล้วแต่ยังหิวอยู่ เค้าสามารถเทเพิ่มได้ถ้าเราสามารถคงคุณภาพของซีเรียลเอาไว้ได้ ผู้ใช้ก็น่าจะพึ่งพอใจอยู่และอาจจะเทอีกซักจานหรือกลับมากินอีกวันพรุ่งนี้ ถ้าเป็นแบบนั้นจริงก็หมายความว่าเราสามารถส่งมอบคุณค่าให้ผู้ใช้ได้แล้ว ส่วนจะเก็บตังหรือเปล่าก็อีกเรื่องนึงนะครับ :-D

Data center คือคลังสินค้าที่ทำให้ผู้ใช้ได้รับคุณค่าอย่างต่อเนื่อง เพราะผู้ใช้มีจำนวนมากคลังสินค้าที่ดีเท่านั้นที่จะส่งมอบสินค้าได้ครบถ้วนตามที่ผู้ใช้ต้องการ ยิ่งเราอยู่ในยุคของ Software as a service มันยิ่งเป็นเรื่องสำคัญมากๆ ที่ต้องทำ Data center ให้ดีพร้อมสำหรับผู้ใช้

หวังว่าเราจะเริ่มเข้าใจความแตกต่างระหว่าง UI กับ UX กันมากขึ้นนะครับ และเราจะเห็นว่าไม่เฉพาะส่วนของ UI ที่มีส่วนสำคัญต่อ User Experience แต่องค์ประกอบอื่นๆ ที่ผู้ใช้ได้สัมผัสล้วนมีผลต่อประสบการณ์ของผู้ใช้ทั้งสิ้นครับ
